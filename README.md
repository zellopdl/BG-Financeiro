# BG - Borda da Grota

Sistema web de gestão financeira para operação rural.

## Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase
- React Query

## Executar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Testes

```bash
npm run test
```

## Estrutura principal

- `src/pages`: telas do sistema
- `src/components`: formulários e componentes de interface
- `supabase/migrations`: migrations SQL

## Marca

Este projeto utiliza a marca **BG - Borda da Grota**.
